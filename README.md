# Posters UI 自动化测试

基于开源电商项目 [Posters Demo Store](https://github.com/Xceptance/posters-demo-store) 的 UI 自动化测试框架，使用 Selenium WebDriver + TestNG + Page Object 模式构建。

## 技术栈
- Java 21
- Selenium WebDriver 4.25
- TestNG 7.7
- Maven
- Page Object 设计模式

## 环境要求
- JDK 21
- Chrome 浏览器（版本 114 或更高）
- Maven 3.6+

## 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/QIQiok/posters-ui-tests.git
cd posters-ui-tests
